Endpoint Usuario
- Crear Usuario  POST/usuario
Descripción:
	Permite registrar un nuevo usuario en el sistema. El endpoint recibe los datos necesarios para la creación de la cuenta, valida la información ingresada y almacena el usuario en la base de datos.
Proceso:
1. El frontend envía los datos del usuario.
2. El sistema valida que los campos obligatorios estén completos.
3. Verifica que el correo o nombre de usuario no exista.
4. Genera la contraseña cifrada.
5. Registra el usuario en la base de datos.
6. Retorna la información del usuario creado.
